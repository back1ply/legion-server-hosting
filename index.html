Frequently Asked Questions

Common questions about Legion Hosting, answered based on their actual service.

Does Legion Hosting offer a free trial?

Yes. Legion Hosting offers a free 48-hour trial for all supported game servers. No credit card is required. You get full access to the game panel, mod installers, and all features during the trial period, so you can test performance before committing to a plan.

How much does Legion Hosting cost?

Pricing starts at $1.58/mo for lightweight games like Terraria and Among Us. Minecraft Java and CS:GO from $3.16/mo, Valheim and 7 Days to Die from $4.72/mo, Rust, Project Zomboid, and ARK from $7.80/mo, and ARK: Survival Ascended from $12.33/mo. Annual billing gives additional savings; new customers can use promo code NEW2026 for 20% off their first order.

What features are included with every plan?

All plans include: no player caps, full FTP access, browser-based file manager, game swapping between supported games, one-click mod installers, database manager, live console, API access, automated schedules (restarts, backups, commands), 24/7 support, always-online servers (runs when you're offline), DDoS protection, and AMD Ryzen processors.

Can I install mods on my server?

Yes. Legion Hosting provides one-click mod installers from the control panel, plus full FTP access and a browser file manager for manual mod installation. Supported setups include Minecraft (Forge, Fabric, NeoForge, Spigot, Paper, Purpur), Terraria (tModLoader), Valheim (BepInEx/Valheim Plus), ARK mods, and custom configs for any game with server-side modding.

Where are Legion Hosting servers located?

Legion Hosting has servers in 5 locations: United States, Canada, Germany (Falkenstein), Australia, and Brazil (São Paulo). All locations include DDoS protection and run on AMD Ryzen hardware.

Can I upgrade or downgrade my server?

Yes. You can upgrade or downgrade your plan at any time from the client area. When upgrading, you pay only the prorated difference. Downgrades are also handled through the self-service panel.

What payment methods are accepted?

Legion Hosting accepts PayPal and Stripe (which includes credit/debit cards, Apple Pay, and Google Pay depending on your device).
